# priority
A React Vue.js / Express.js web application for a fuzzy academic prioritization system 

# Barebones Overview
Automatically creates a priority schedule for your upcoming deadlines according to importance.

# Instructions to run
* CD into ./server/src
* Run "npx nodemon app.js"
* CD into ./client/src
* Run "npm run dev"
